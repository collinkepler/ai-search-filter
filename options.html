Rules

Each rule is one topic. Block prevents matching content. Allow force-allows content that would otherwise be blocked. Only allow turns on allowlist mode: as soon as any "Only allow" rule exists, everything not matching one is blocked by default. Block rules still subtract from that set; Allow rules still override blocks.
Click the scope chip below each rule to limit it to specific hostnames (e.g. youtube.com). Scoped rules aren't sent to Claude on other sites — faster and cheaper. On a site no Only-allow rule scopes to, default-allow takes over.
The switch on the left of each rule disables it without deleting — disabled rules aren't sent to Claude at all. Disabling a Block or Only-allow rule needs approval; disabling an Allow rule does not.

Filtering layers

Each layer is independent. Turn on whichever combination works for you.
Layer 0 — Page-level classification
Always on. Every page navigation is classified by URL + title + meta description.
Layer 2 — Site-specific NSFW CSS (Reddit, Twitter, YouTube)
Hides content these sites already tag as NSFW or sensitive. Zero API cost, instant. Catches most porn previews in feeds.
Layer 1 — NSFW image scanner
Every image > min size is blurred immediately on render; the blur clears for safe images and the image is hidden entirely for NSFW (porn or suggestive). Eliminates fast-scroll flashes. Costs ~$0.0005/image (cached 30 days).
One small Claude call per new site, cached 30 days. Saves API cost on safe sites without manual exclude-list maintenance.
One per line. Subdomains included. Saves API cost on sites where image checks aren't important.
Layer 3 — Feed post text classifier
On Reddit, Twitter, YouTube: extracts post titles + channels, batches them, classifies via Claude. Catches text-flagged content even when not NSFW (gambling discussions, drama, etc.).

Usage limits

Limit how much you use specific parts of the web each day or week. Three modes:
Visit count — block after N distinct page opens (e.g. 1 Netflix episode/day).
Time on page — block after N minutes while the tab is visible (e.g. 30 min/day on Reddit).
AI-match override — let up to N AI-blocked pages through per period, then enforce the block.
Patterns: netflix.com/watch/*  |  *.reddit.com/*  |  youtube.com/watch*  |  x.com/*

Always-allow domains

Skip all checks (including image/post scanners) for these. One per line. Subdomains included.
